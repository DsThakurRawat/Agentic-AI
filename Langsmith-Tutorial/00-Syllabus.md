# LangSmith Masterclass Syllabus

This syllabus is structured around the core video curriculum to ensure we cover all critical aspects of LangSmith, from basic tracing to advanced LangGraph integrations.

## Module 1: Core Fundamentals & Observability
- **Setting up Agenda**
- **Example 2** (Introductory use case)
- **Observability** (Why do we need observability in GenAI?)
- **What does LangSmith Trace?** (Inputs, outputs, internal agent monologue)
- **Core Concepts** (Runs, Traces, Datasets, Evals)

## Module 2: Applied Code Demonstrations
- **Code Demo 1:** Basic environment setup and LangChain tracing implementation.
- **Code Demo 2:** Instrumenting custom functions and tracking token costs.

## Module 3: Advanced Application Tracing
- **Tracing RAG App:** Tracking document retrieval, embedding generation, and final answer synthesis.
- **Testing Query:** Debugging edge cases and refining prompts for RAG pipelines.

## Module 4: Agentic Frameworks & Production Features
- **LangGraph + LangSmith Integration:** Tracing cyclical state machines, multi-agent handoffs, and human-in-the-loop approvals.
- **Features of LangSmith:** Deep dive into the Prompt Hub, automated LLM-as-a-judge evaluations, and dataset management.
- **Outro:** Summary and next steps for production deployment.
