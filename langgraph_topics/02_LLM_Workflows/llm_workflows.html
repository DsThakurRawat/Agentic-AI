Orchestration Topologies

LangGraph acts as a programmable compiler for multi-stage applications. Developers map control flow logic directly onto explicit design topologies based on computational requirements.

Topology A

Prompt Chaining

graph LR A([S]) --> B[Plan] --> C[Expand] --> D([E])

Linear additive string payload overrides where upstream planner nodes emit initial structures.

Topology B

Dynamic Routing

graph TD A[Classify] --> R{?} R --> B[Tech] R --> C[Bill]

Scalar conditional literal mappings targeting independent downstream nodes based on triage.

Topology C

Fan-Out / Fan-In

graph TD A[S] --> B[W1] A --> C[W2] B --> D[Join] C --> D

Concurrent execution channels aggregated via custom state reducers to avoid clobbering.
