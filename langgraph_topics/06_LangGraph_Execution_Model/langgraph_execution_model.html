02 The Persistence Layer

By attaching a `MemorySaver` checkpointer during compilation, LangGraph automatically serializes every state transition. This allows for **Time Travel** and long-running conversation threads. 

from langgraph.checkpoint.memory import MemorySaver

memory = MemorySaver()
app = graph.compile(checkpointer=memory)

# Persistent Thread Execution
config = {"configurable": {"thread_id": "session_01"}}
app.invoke(input, config=config)

Why thread_id?

The `thread_id` is the key that unlocks specific historical snapshots. Without it, the application runs in a transient, one-off mode. With it, the agent has "Long-term" memory within a specific session context.
