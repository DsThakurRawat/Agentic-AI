01 Theoretical Foundations

Corrective RAG (CRAG)

Focuses on **Retrieval Robustness**. If internal documents are ambiguous or irrelevant, the agent triggers a fallback to Web Search (e.g., Tavily) to repair the context.

Retrieval Grader

Evaluates context quality before it reaches the generation phase.

Self-RAG (Generation Fidelity)

The agent acts as its own "Judge." It uses specialized critique tags to evaluate if every claim in its response is supported by the retrieved context.

  • Is-Relevant: Does context answer the query?
  • Is-Supported: Hallucination check.
